## 1. Use MDX (Required)

`.md` cannot `import` components. Use `.mdx` (e.g. `src/content/blogs/<slug>/index.mdx`).

## 2. User Component Examples

<Aside type='tip' title='Tip'>
You can render theme components directly inside MDX.
</Aside>

<Tabs>
  <TabItem label='A'>
    Panel A content
  </TabItem>
  <TabItem label='B'>
    Panel B content
  </TabItem>
</Tabs>

<Spoiler label='Click to reveal'>
Hidden content goes here.
</Spoiler>
