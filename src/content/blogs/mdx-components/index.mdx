import { GithubCard, LinkPreview, QRCode, ImageGroup, WebVideo } from '@/components/advanced'

## 1. 必须使用 MDX

`.md` 文章无法 `import` 组件；请使用 `.mdx`（如 `src/content/blogs/<slug>/index.mdx`）。

## 2. User 组件示例

<Aside type='tip' title='Tip'>
你可以在 MDX 中直接使用主题组件。
</Aside>

<Tabs>
  <TabItem label='A'>
    A 面板内容
  </TabItem>
  <TabItem label='B'>
    B 面板内容
  </TabItem>
</Tabs>

<Spoiler label='点击展开'>
隐藏内容写在这里。
</Spoiler>
